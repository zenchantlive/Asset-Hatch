# Suggested Commands

## Development
- Start Dev Server: `bun dev` (or `npm run dev`)

## Build
- Build Production: `bun run build`

## Testing
- Run Tests: `bun run test`
- CI Test: `bun run test:ci`

## Linting
- Lint Code: `bun run lint`

## Formatting
- No explicit format script found, likely Prettier or ESLint handles this.
